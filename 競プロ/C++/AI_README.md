# AIが読み込むファイル

## 依頼する時のプロンプト

```md
日付 = 2025_mm_dd_*

{日付} のついてるファイルを `競プロ/C++` から読み取って、以下の内容に沿って要約し、ルールに従いObsidianに書き込んでください。
```

## Obsidianのフォルダの位置

`~/Documents`

## Obsidianに作成するファイル名

`yyyy年mm月dd日`

## 要約するファイル

Path: `競プロ/C++`
読み取るファイル: 指定された日付のファイル全て (_1, _2などでその日のやったことを分割しています。)

## 記載する内容のテンプレート

- やったことの要約
  - 学んだアルゴリズム
  - アルゴリズムの説明の要約
  - 他に学んだと思われる内容
  - どういった問題を解いたか
  - 最後にユーザーが感想を書く欄
